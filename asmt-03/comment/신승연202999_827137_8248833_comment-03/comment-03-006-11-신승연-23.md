# 과제-03 단문 연습에 대한 코멘트

- 대상과제: `과제-03 단문 연습`
- 코멘트를 제공하는 학생: `006-11 신승연` 
- 코멘트를 받는 학생: `006-23 강서연` 

---

## 코멘트

현재 제출된 OREO 논증문은 우선, 논제 진술(Opinion)이 명확하게 제시되고 있지 않다. '다양한 자극과 유혹 속에서 이루어지는 소비 행위가 개인의 자율적 의사결정이라고 말할 수 있는가?'라는 첫 문장에서 이후 바로 이유(Reason)로 넘어가서 자신이 주장하고자 하는 바가 무엇인지 분명히 드러나지 않는다. "개인의 자율적 의사결정이 아니다"라고 말하고 싶은 것인지, "온전히 개인의 자율적인 의사결정이라고 할 수는 없다"라고 말하고 싶은 것인지 정확히 알 수가 없다. 또한 제시된 이유(Reason)와 주장 간의 논리적 연결성 역시 약한 측면이 있다. 특히 '알고리즘은 소비자들이 과거 데이터를 바탕으로 개인의 취향에 더 잘 들어맞는 집단을 접하도록 하면서'라는 부분에서 그 논리적 취약성이 드러난다. 해당 문장은 물론 알고리즘으로 인해 사람들이 타인의 소비에 더 많이 노출되고, 이것이 경쟁적 소비로 이어진다는 뜻으로 쓰여졌을 것이다. 하지만 '과거의 데이터'나 '개인의 취향'이라는 표현들은 비록 알고리즘에 의해 추천되었어도, 그것이 개인의 선호를 반영한다면 이에 따른 소비는 개인의 자율적 의사결정에 더 가깝지 않나라는 주장 (글쓴이의 주장과 반대 주장)을 뒷받침할 수 있을 것이다. 더 나아가 왜 알고리즘, 마케팅, SNS 등이 단순히 '영향'이 아닌 '강요'로서 작용하여 개인들의 소비 선택이 자율적인 의사결정이 아니게 되는 수준까지 이어지는지에 대해서도 구체적인 근거를 제시할 필요가 있다.

## 체크 포인트

### **구성요소**
- [x] 논제 진술문을 찾거나 식별하기 어렵다.
- [ ] 논증 진술문을 찾거나 식별하기 어렵다.
- [ ] 증거/사례 진술문을 찾거나 식별하기 어렵다.
- [ ] 논제 재-진술문을 찾거나 식별하기 어렵다.
- [ ] 논제, 논증, 증거/사례, 논제 재-진술문 각각 기능적으로 명확히 구분되어 진술되었다.

### **논제 진술의 명확성 및 표현** 
- [x] 논제 진술문이 너무 일반적이거나 모호하여 독자가 핵심 주장을 명확히 이해하기 어렵다. 
- [x] 논제 진술문의 핵심 용어 선택이 모호하거나, 애매하거나, 잘못되었다. 
- [ ] 논제 진술문이 필자의 의도를 명확하게 전달하고 있다. 

### **논증과 증거/사례 제시** 
- [ ] 이유(Reason)가 논제와 무관하다.
- [x] 이유(Reason)가 논제를 충분히 뒷받침하지 못하고 있으며, 명확한 논리적 연결이 부족하다. 
- [ ] 증거(Example)가 구체적이지 않거나, 논제 및 이유와의 관계가 명확하지 않다. 
- [ ] 증거(Example)가 사실적 근거보다는 개인적 견해에 의존하고 있다. 
- [ ] 이유와 증거가 논리적으로 명확하게 연결되었다. 

### **논제 재-진술** 
- [ ] 논제 재-진술문이 적절하지 않거나, 단순한 반복에 그친다. 
- [x] 논제 재-진술문이 단락 도입 부분의 논제 진술을 보다 구체화, 명료화하여, 단순한 반복을 넘어서는 역할을 수행한다.
