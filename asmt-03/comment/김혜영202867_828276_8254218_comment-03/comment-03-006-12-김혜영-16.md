# 과제-03 단문 연습에 대한 코멘트

- 대상과제: `과제-03 단문 연습`

## 코멘트

현재 제출된 OREO 논증문은 논제 진술문, 이유 진술문, 증거/사례 진술문, 논제 재-진술문을 각각 기능적으로 명확히 구분하여 진술해주어 OREO 모델의 구성요소를 충분히 충족하고 있다. 우선, 논제 진술(Opinion)은 필자의 의도를 독자에게 명확하게 전달하고 있어, 글의 초점을 이해하기 쉬웠다. 다만, 이유(Reason) 진술문은 논제를 논리적으로 뒷받침하지 못하고 있다. 교화 또한 범죄를 예방하는 효과를 지니기에, 단지 '처벌이 범죄를 예방하는 효과를 가진다'는 진술문은 논제 '처벌을 교화보다 우선시해야 한다'를 논리적으로 뒷받침하지 못한다. 증거(Example)로는 Ashna Arora의 연구 결과를 들며 이유 진술문을 잘 뒷받침해주고 있지만, 이유와 논제 사이의 논리적 연결관계가 어색하기에 차라리 증거를 바꿔준다면 좋을 것 같다. '범죄를 예방하고 범죄 행위를 저지를 가능성이 큰 청소년을 교화 및 보호하는 것이 범죄를 저지른 이후의 교화보다 더 효과적이기 때문이다.'가 이유 진술문이 되고, 이를 뒷받침하는 증거(Example)를 들었다면 논제-이유-증거의 세 부분이 논리적으로 연결된 논증문이 될 수 있었을 것 같다. 의견 재진술(Opinion Restated) 부분은 초기 논제 진술을 보다 구체화, 명료화하여, 단순한 반복을 넘어서 효과적으로 주장을 강조하고 있다.

## 체크 포인트

### **구성요소**
- [ ] 논제 진술문을 찾거나 식별하기 어렵다.
- [ ] 논증 진술문을 찾거나 식별하기 어렵다.
- [ ] 증거/사례 진술문을 찾거나 식별하기 어렵다.
- [ ] 논제 재-진술문을 찾거나 식별하기 어렵다.
- [x] 논제, 논증, 증거/사례, 논제 재-진술문 각각 기능적으로 명확히 구분되어 진술되었다.

### **논제 진술의 명확성 및 표현** 
- [ ] 논제 진술문이 너무 일반적이거나 모호하여 독자가 핵심 주장을 명확히 이해하기 어렵다. 
- [ ] 논제 진술문의 핵심 용어 선택이 모호하거나, 애매하거나, 잘못되었다. 
- [x] 논제 진술문이 필자의 의도를 명확하게 전달하고 있다. 

### **논증과 증거/사례 제시** 
- [ ] 이유(Reason)가 논제와 무관하다.
- [x] 이유(Reason)가 논제를 충분히 뒷받침하지 못하고 있으며, 명확한 논리적 연결이 부족하다. 
- [ ] 증거(Example)가 구체적이지 않거나, 논제 및 이유와의 관계가 명확하지 않다. 
- [ ] 증거(Example)가 사실적 근거보다는 개인적 견해에 의존하고 있다. 
- [x] 이유와 증거가 논리적으로 명확하게 연결되었다. 

### **논제 재-진술** 
- [ ] 논제 재-진술문이 적절하지 않거나, 단순한 반복에 그친다. 
- [x] 논제 재-진술문이 단락 도입 부분의 논제 진술을 보다 구체화, 명료화하여, 단순한 반복을 넘어서는 역할을 수행한다.
